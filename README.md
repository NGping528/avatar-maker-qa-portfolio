# 捏臉網站 QA 測試作品集

## 專案簡介

本專案為一個角色捏臉網站，使用者可以透過不同素材分類選擇角色部件，組合出不同角色外觀。

網站目前包含角色素材選取、角色圖層顯示、隨機組合、下載圖片與素材設定管理等功能。

## 專案目標

此作品集主要用於展示 QA 測試思維與測試文件撰寫能力，包含：

- Test Case 測試案例
- Bug Report 缺陷回報
- UI / UX 優化建議
- 功能建議
- 規格確認
- 回歸測試紀錄

## 測試範圍

本次測試範圍包含：

- 角色素材選取功能
- 角色素材圖層順序
- 隨機組合功能
- 新增素材後隨機功能是否可抽取新素材
- 下載圖片功能
- UI / UX 操作體驗
- 素材資料管理流程

## 測試案例

### TC-006 新增素材後隨機組合功能是否可抽取新素材

測試目的：

確認新增素材後，既有隨機組合功能是否能正確讀取新素材。

測試結果：

新增素材成功顯示於 hair 分類中，並可被隨機組合功能抽取，狀態為 Pass。

測試價值：

此案例屬於功能測試與回歸測試，可確認新增素材後未影響既有功能。
